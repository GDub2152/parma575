PARMA575 PROFESSIONAL WEBSITE
=============================

How to use:
1. Unzip this folder.
2. Open index.html in your browser.
3. Upload all files/folders to your web host or GitHub Pages.
4. Replace placeholder images in assets/images with your own photos.
5. Add PDFs and documents to the docs folder.
6. Edit the HTML pages to update events, repeater information, and club news.
